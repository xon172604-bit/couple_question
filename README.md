Question Tackle 🎮

Question Tackle is a small browser game where players answer questions to progress. The idea behind the game is simple: combine quick thinking with gameplay. Instead of just clicking through questions, the player interacts with the game world while answering them.

This project was built as a simple experiment to explore game development and interactive learning. The goal was to make something lightweight that can run directly in the browser without requiring downloads or complicated setup.

How the Game Works

When the game starts, the player is presented with a question. The player must choose the correct answer to move forward. Each correct answer allows the player to continue, while wrong answers may slow progress or reset the challenge depending on the level.

The gameplay focuses on keeping things simple and engaging rather than complex mechanics. The main aim is to make answering questions feel more like playing a game rather than solving a quiz.

Features
	•	Simple and interactive gameplay
	•	Question-based progression
	•	Lightweight and fast loading
	•	Runs directly in the browser
	•	Easy to modify questions and game logic

Technologies Used

This project was built using:
	•	Python
	•	Flask
	•	HTML
	•	CSS
	•	JavaScript

Flask is used to serve the game and handle the basic backend logic, while the frontend is responsible for displaying the questions and handling user interaction.
Future Improvements

Some ideas for improving the game include:
	•	Adding more questions and levels
	•	Introducing a scoring system
	•	Adding animations and sound effects
	•	Saving player progress
	•	Multiplayer or competitive modes

Why This Project Exists

This game was created mainly for learning and experimentation. It helped explore how web technologies and simple backend frameworks can be used together to create interactive experiences.

Even though it is a small project, it demonstrates how quickly a simple game can be built using web tools.
